Comandos Bash

-a -----> mostra os arquivos ocultos 

. -----> referência da pasta para ela mesma, mostra onde você está 

.. ----> vai pra pasta mãe do diretorio onde estou

ls ----> lista oque existe no dirétorio 

cd -----> muda de diretório

pwd -----> mostra o caminho do diretório

mkdir ------> serve para criar uma pasta no diretório onde estou

cat -------> mostra oque tem dentro do meu arquivo que eu consiga editar

rm -----> remove arquivo ou diretório

git init . ------> cria um repositório vazio dentro do meu diretório

rm -rf ----> remove de forma recursiva e forçada todos os diretórios
